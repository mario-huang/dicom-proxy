# dicom-proxy
DICOM DIMSE proxy.

## Supported DIMSE services
- C-Echo
- C-Find
- C-Move

## Features
* simple to use
* docker support

## What is it for?
If you only have a single AE (for example, from a hospital) but have multiple applications that need to access the PACS system, this proxy tool can help you.
